# aws-s3-project# AWS S3 Bucket Project

## 📌 Objective

To create an S3 bucket and upload files using AWS.

## 🛠️ Tools Used

* AWS S3
* Web Browser

## 🚀 Steps

1. Login to AWS Console
2. Go to S3 service
3. Click "Create Bucket"
4. Enter bucket name and region
5. Disable/enable public access as needed
6. Click Create

## 📤 Upload File

1. Open bucket
2. Click Upload
3. Add files
4. Click Upload
